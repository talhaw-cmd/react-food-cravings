# 🍴 Crave Meals – React Food Discovery App

Crave Meals is a playful and modern food discovery web app built with React.  
It allows users to search meals in real time, explore categories and cuisines,  
and watch full recipe videos using data from **TheMealDB API**.

---

## 🚀 Features

- 🔍 Real-time meal search
- 🍽 Browse meals instantly by name
- 🌍 Displays meal category and origin (area)
- ▶ Watch full recipe videos on YouTube
- 🎨 Playful, clean, and modern UI
- 📱 Fully responsive design
- ⚡ Fast and lightweight React app

---

## 🛠 Tech Stack

**Frontend**
- React.js (Functional Components)
- React Hooks (`useState`, `useEffect`)

**API**
- TheMealDB API

**HTTP Client**
- Axios

**Styling**
- CSS3 (Custom playful UI)

---

## ⚙ How It Works

1. User types a meal name in the search bar  
2. App fetches data from TheMealDB API  
3. Matching meals are displayed in a grid  
4. Each meal card shows:
   - Meal image  
   - Meal name  
   - Category & area  
   - Recipe video button
